
## Planned Changes

This project is still under active development. Below is a list of planned features and improvements.
Items are listed in no particular order and may be implemented incrementally over time.

> This roadmap represents planned features, not guarantees.
> Implementation details may change as the project evolves.
---

### Fix Logic Diagrams
- Correct existing logic diagram issues

---

### Create Missing Diagrams
- Add diagrams for:
  - Register File
  - Program Counter (PC)
  - Control Unit
  - Sign Extension
- Potentially add Instruction Cache and Data Cache diagrams

---

### Implement Data Cache
- Implement backend support for a Data Cache
- Integrate the Data Cache into the visualizer
- Add memory instructions such as `lw` and `sw`
- Allow loading custom values into the Data Cache

---

### Shifter Diagram and Signals
- Create a shifter diagram within the ALU

---

### Code Editor Improvements
- Fix visual issues in the code editor
- Improve backend parsing and error handling

---

### Binary Values
- Add support for binary literals in the code editor
- Display wire values in binary format where applicable

---

### Register Values Menu
- Add a menu displaying register values per cycle
- Feature is experimental and subject to revision

---

### ASCII Support
- Add the ability to display ASCII representations of values
- Allow loading ASCII data into the Data Cache

---

### Dark Mode
- Add a dark mode option across the entire website
- Ensure diagrams and wire values remain readable in both themes
